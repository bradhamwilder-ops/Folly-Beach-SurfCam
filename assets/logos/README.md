Place sponsor logos in this folder.

Recommended formats:
- SVG for the cleanest result
- Transparent PNG if SVG is not available

Recommended naming:
- `era-wilder-realty.svg`
- `five-palms-media.png`

Then update `logoPath` in `/Users/bradhamwilder/Bradham's Codex File/Washout-SurfCam/script.js`, for example:

`logoPath: "./assets/logos/era-wilder-realty.svg"`

Tips:
- Avoid screenshots or logos pulled from websites
- Avoid JPG when the logo needs transparency
- Use art that is at least 2x the displayed size if using PNG
