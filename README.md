# Washout-SurfCam

Multi-page static website concept for a Folly Beach Washout livestream, fishing content, real estate visibility, rental partners, Folly history, and local advertising outreach.

## Quick start

1. Open `index.html` in a browser.
2. Edit `script.js` and replace `streamEmbedUrl` with your real livestream embed URL.
3. Add or change sponsor website URLs for:
   - Bradham Wilder / ERA Wilder Realty
   - Five Palms Media
4. Use the additional pages:
   - `fishing.html`
   - `real-estate.html`
   - `stay.html`
   - `history.html`
   - `advertise.html`

## Notes

- The current embedded video is a placeholder.
- Fishing regulation snapshots should always be re-checked against SCDNR before publishing live updates.
- This site is static, so it can be hosted on Netlify, Vercel, GitHub Pages, or any web server.
